# README.md

# DiskAutoRecord — Mac 外接硬盘自动归档可视化脚本

**一款专为 Obsidian 优化的全自动硬盘目录生成工具**

无需手动操作、无需插件、纯 Python 原生运行，自动扫描外接硬盘、生成**层级缩进、彩色层级、原生折叠、结构完全正确**的可视化目录文档。

---

## ✨ 项目亮点（最终版特性）

- **全自动后台扫描**：常驻后台定时刷新所有外接硬盘目录

- **热插拔监听**：硬盘拔出前可手动选择最后一次归档备份

- **完美嵌套折叠**：父文件夹折叠自动收纳所有下级目录/文件

- **层级视觉系统**：纯方形色块分级图标 + 逐级缩进，结构一目了然

- **智能自动折叠规则**：控制目录默认展开/折叠状态

- **精准递归统计**：文件夹真实总大小 + 项目数量统计

- **Obsidian 原生兼容**：无 JS 冗余、无第三方插件依赖、渲染零错乱

- **极简干净输出**：无多余按钮、无全局收纳、无冗余功能

## 📌 功能完整清单

### 1. 硬盘扫描与监控系统

- 自动读取 Mac `/Volumes` 下所有外接挂载硬盘

- 自动过滤系统盘，仅处理外接存储设备

- 后台定时全自动刷新目录结构

- 硬盘拔出事件监听，支持拔出前最后一次归档备份

- 自动过滤系统垃圾目录/文件（.DS_Store、__pycache__ 等）

### 2. 目录数据统计能力

- **递归计算文件夹真实总大小**（包含所有子文件、子目录）

- 统计每个目录总项目数（子文件夹 + 文件总数）

- 仅文件夹展示【大小 / 项目数】，不展示冗余时间信息

- 文件展示【大小 / 修改时间】，信息精准区分

### 3. 层级视觉展示系统（核心特色）

- **纯正方形色块层级图标**（无爱心、无圆形、无异形）

- 10级固定配色循环，层级色差清晰，长期多层目录不混乱

层级配色规则：

```Plain Text
0级（根目录）→ 📁 文件夹
1级 → 🟦 蓝色方块
2级 → 🟩 绿色方块
3级 → 🟧 橙色方块
4级 → 🟥 红色方块
5级 → 🟪 紫色方块
6级 → 🟨 黄色方块
7级 → 🟫 棕色方块
8级 → ⬛ 深灰方块
9级 → ⬜ 浅灰方块

```

- **严格逐级缩进**：每加深一级，自动缩进，层级结构绝对对齐

- 文件自动匹配类型图标（图片/视频/音频/文档/压缩包/程序等）

- 右侧信息统一右对齐，排版整洁规整

### 4. 智能折叠展开系统（最终稳定版）

完全使用 HTML 原生 `details/summary`，无自定义 JS、无全局按钮

- **原生嵌套行为正常**：父文件夹折叠 → 所有下级全部收纳

- 每个文件夹支持**独立手动展开/收起**

- 智能自动折叠规则：
  

    - 目录深度 ≥5 级 → 默认折叠

    - 单目录项目数量 >10 项 → 默认折叠

    - 浅层级、少项目目录默认展开，方便预览

- 彻底移除早期所有「一键收纳/一键展开」冗余功能，界面干净纯粹

### 5. 文档输出规范

- 每个硬盘独立生成对应 `.md` 文件

- 头部包含：硬盘名称、挂载路径、最新更新时间

- 保留原始标题代码行但注释隐藏，兼顾规范与美观

- 底部输出统计信息：总文件数量、扫描耗时

## 🛠 技术难点 & 最终解决记录

本次开发解决了大量 Obsidian 渲染 + Python 遍历的经典冲突问题，为最终稳定版核心基石：

- **解决：details 折叠与空格缩进冲突**
Obsidian 原生会吃掉折叠标签前置空格，改用 CSS 边距实现层级缩进，折叠、缩进二者完全兼得


- **解决：Python3.9 不支持 match/case 语法报错**
重写图标匹配逻辑，使用通用 if/elif 兼容低版本 Python


- **解决：父文件夹无法收纳下级文件夹**
重构目录生成逻辑，抛弃平铺结构，采用**栈结构维护标准 HTML 嵌套层级**，实现原生父子收纳行为


- **解决：层级错位、缩进混乱、子目录丢失**
修复栈闭合判断逻辑，修正 os.walk 深度优先遍历导致的结构断裂问题


- **解决：自动展开目录无法手动收起**
优化 open 属性渲染逻辑，完全交由浏览器原生控制状态


- **清理所有冗余功能**
移除全部自定义按钮、层级控制 JS、全局收纳功能，还原最稳定原生渲染

## ⚙️ 可自定义配置项

所有核心规则均可在代码头部直接修改：

- `SCAN_INTERVAL`：后台自动扫描间隔

- `SAVE_FOLDER`：MD 文档输出路径

- `MAX_EXPAND_DEPTH`：最大自动展开层级（超过自动折叠）

- `MAX_ITEM_TO_EXPAND`：单目录最大展开项目数（超过自动折叠）

- `EXCLUDE_DIR_LIST`：自定义忽略扫描的目录名单

- `FILE_ICON_MAP`：自定义文件类型图标映射
